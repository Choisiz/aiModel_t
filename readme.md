# 머신러닝 학습기록

## KNearest

### k_nearest.py
- **지도학습 알고리즘** 기본 구현
- **샘플링편향** 문제 발생
- 도미/빙어 분류 (49개 샘플)

### k_nearest2.py  
- **훈련세트/테스트세트** 분리
- **데이터 전처리** (넘파이 활용)
- **샘플링편향** 해결 (무작위 셔플링)

## 주요 개념
- KNN 알고리즘
- 이진분류 (도미:1, 빙어:0)  
- 데이터 분할의 중요성
- 모델 평가

## 사용 라이브러리
- sklearn (KNeighborsClassifier)
- numpy (배열 처리)
- matplotlib (시각화)
